Mengapa menggunakan CSS dan JavaScript dalam pemrograman web?

CSS digunakan untuk mengatur tampilan dan layout halaman web. JavaScript memberi interaktivitas seperti validasi form atau AJAX.

Bagaimana CSS dan JavaScript dapat diintegrasikan dalam pemrograman web?

CSS:
- Inline CSS: <p style="color: blue;">Teks</p>
- Internal CSS: dalam tag <style>
- External CSS: <link rel="stylesheet" href="style.css">

JavaScript:
- Inline JS: <button onclick="alert('Hello!')">
- Internal JS: dalam tag <script>
- External JS: <script src="script.js">

Jelaskan apa saja keuntungan menggunakan JavaScript dan CSS?

Keuntungan JavaScript: Interaktivitas, validasi, AJAX, aplikasi kompleks.
Keuntungan CSS: Pisahkan konten & tampilan, lebih cepat, responsif, hemat waktu.

Jelaskan apa saja keuntungan menggunakan Bootstrap?

Bootstrap mempermudah pembuatan halaman web responsif dengan komponen siap pakai, kompatibel lintas browser, dan banyak dokumentasi.

Contoh penggunaan Bootstrap dan penjelasannya

<button class="btn btn-primary">Klik Saya</button>

Class btn btn-primary untuk tombol biru, container mt-5 memberi margin dan padding.
